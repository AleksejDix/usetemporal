### Getting Started Update Pattern

```markdown
# Getting Started

## Choose Your Integration

useTemporal offers three ways to use the library:

### Vue.js Application

```bash
npm install @allystudio/usetemporal-vue
```

```typescript
import { useTemporal, usePeriod } from '@allystudio/usetemporal-vue'
import { createNativeAdapter } from '@allystudio/usetemporal-vue'

const temporal = useTemporal({
  adapter: createNativeAdapter(),
  date: ref(new Date())
})

const month = usePeriod(temporal, 'month')
```

[Learn more →](/guide/vue)

### React Application

```bash
npm install @allystudio/usetemporal-react
```

```typescript
import { useTemporal, usePeriod } from '@allystudio/usetemporal-react'

function MyComponent() {
  const temporal = useTemporal({
    adapter: createNativeAdapter(),
    date: new Date()
  })

  const month = usePeriod(temporal, 'month')
}
```

[Learn more →](/guide/react)

### Vanilla JavaScript (No Framework)

```bash
npm install @allystudio/usetemporal
```

```typescript
import { period, divide } from '@allystudio/usetemporal/operations'
import { createNativeAdapter } from '@allystudio/usetemporal/native'

const adapter = createNativeAdapter()
const month = period(adapter, new Date(), 'month')
const weeks = divide(adapter, month, 'week')
```

[Learn more →](/guide/core)
```

### Framework Comparison Table

```markdown
| Feature | Core | Vue | React |
|---------|------|-----|-------|
| Bundle Size | ~8KB | ~22KB | ~20KB |
| Reactivity | No | Yes (Vue Refs) | Yes (useState) |
| Use Case | Static/Server | Vue apps | React apps |
| API Style | Pure functions | Composables | Hooks |
```
